# CLAUDE.md

Контекст для кодового агента. Читай перед любыми изменениями.

## Проект

Сайт-визитка дизайнера интерьеров **Натальи Селезнёвой** — одностраничник с портфолио, услугами и формой заявки. Целевая аудитория: клиенты с бюджетом от 5 млн ₽.

## Стек

- **React 19 + Vite 8** — фронтенд и сборка
- **Tailwind CSS 4** — стили
- **Swiper 12** — галереи (портфолио, отзывы)
- **React Hook Form** — форма заявки
- **Vercel** — хостинг

## Структура файлов

```
site/
  index.html
  vite.config.js
  package.json
  src/
    main.jsx          # точка входа
    App.jsx           # собирает все секции по порядку
    index.css         # Tailwind + Swiper imports + базовые стили
    App.css           # не используется (шаблонный остаток, не трогать)
    data.js           # реэкспорт — НЕ содержит данных, только индекс
    data/
      projects.js     # portfolioItems — 6 объектов портфолио
      services.js     # services — 3 услуги
      process.js      # processSteps — 4 этапа работы
      reviews.js      # reviews — 4 отзыва
    components/
      Nav.jsx         # навигация (фиксированная, меняет вид при скролле)
      Hero.jsx        # главный экран с фоновым фото
      Portfolio.jsx   # сетка проектов + модальная галерея
      About.jsx       # раздел о дизайнере
      Services.jsx    # три карточки услуг
      Process.jsx     # этапы работы
      Reviews.jsx     # карусель отзывов
      ContactForm.jsx # форма заявки (имя, телефон, email, бюджет, текст)
      Footer.jsx      # подвал
    assets/
      react.svg       # не используется
      vite.svg        # не используется
```

## Важные соглашения

- **Данные** — все статические данные живут в `src/data/*.js`. Компоненты импортируют через `src/data.js` (индекс). Не добавляй данные напрямую в компоненты.
- **Имя дизайнера** — везде **Наталья Селезнёва**. Не менять на плейсхолдеры.
- **Стили** — только Tailwind-классы, кастомный CSS не добавлять без необходимости.
- **App.css** — файл есть, но нигде не импортируется. Не удалять, не импортировать.
- **Форма** — `ContactForm.jsx` сейчас логирует данные в `console.log`. Telegram-интеграция не подключена.

## Что не трогать без явного запроса

- Порядок секций в `App.jsx`
- Цветовую палитру (stone, white, black — премиальный минимализм)
- Шрифт (Georgia, serif — задан в `index.css`)
- Структуру `data/*.js` — если нужно добавить поле, добавляй во все объекты массива
